# 🎭 ShowView - 뮤지컬/연극 좌석 시야 공유 플랫폼

> **"앞열이라서 예매했는데, 막상 가보니 시야가 가리거나 별로였던 적이 있으신가요?"**  
> ShowView는 관객들이 예매 전에 극장 좌석별 실제 시야를 미리 확인하고 실패 없는 예매를 할 수 있도록 돕는 웹 브라우저 기반 플랫폼입니다.

---

## 📝 기획 배경 및 개요 (PRD)

* **Why (기획 의도)**
  * 무대와 가까운 앞열 좌석이라 기대하고 선택했으나, 실제 극장에 착석했을 때 시야 제한이나 무대 높이 등으로 인해 시야가 불만족스러웠던 경험을 해결하기 위함입니다.
* **Who (타깃 사용자)**
  * 연극 및 뮤지컬 공연을 예매하려는 관객 전체.
* **What (핵심 가치)**
  * 극장별 좌석 배치도에서 원하는 좌석을 클릭하면, 해당 좌석에 앉았을 때의 **실제 시야 사진**과 그 자리에서 관람했던 관객들의 **리뷰 평점**을 사전에 확인할 수 있게 합니다.
* **When (사용 시점)**
  * 연극/뮤지컬 티켓 예매 전, 좌석을 결정하고 선택하는 단계에서 주로 사용합니다.
* **Where (제공 환경)**
  * 웹 브라우저 (모바일 및 데스크톱 반응형 웹 지원).
* **How (작동 방식)**
  * 극장 배치도에서 각 좌석별 실제 무대 시야 사진을 확인하고, 시야 만족도에 따른 평균 별점 및 관객들이 남긴 생생한 리뷰 코멘트를 조회/등록할 수 있습니다.

---

## 🛠️ 제공 기능

### 기능1) 극장 혹은 공연 검색
* 등록된 극장 및 상영 중인 공연 리스트를 제공합니다.
* 텍스트 입력창을 통해 원하는 극장 이름 또는 공연명을 실시간으로 검색할 수 있습니다.

### 기능2) 좌석 배치도 조회
* 선택한 극장의 1층, 2층 등 층별 상세 좌석 배치도를 렌더링합니다.
* 우측 상단 보기 설정 필터를 통해 **좌석별 시야 평점 등급(별점 수준별) 색상 오버레이**를 띄워 어떤 구역이 만족도가 높고 낮은지 직관적으로 구별할 수 있습니다.

### 기능3) 좌석 선택과 시야 사진 확인
* 배치도에서 특정 좌석을 클릭하면 해당 좌석 번호와 실제 시야 사진, 만족도 평균 별점 및 관객 리뷰 코멘트 목록이 우측 상세 패널에 표시됩니다.
* 시야 사진을 클릭하면 원본 크기로 볼 수 있는 라이트박스 이미지 뷰어가 활성화됩니다.

### 기능4) 시야 사진 업로드 및 별점 남기기
* 닉네임 입력을 통해 가상 로그인을 진행합니다.
* 로그인 완료 후 원하는 좌석에 시야 만족도 별점(1~5점), 상세 시야 설명 코멘트 작성 및 사진을 직접 첨부하여 등록할 수 있습니다.
* 등록한 파일은 Base64 데이터 스트링으로 변환되어 브라우저 로컬 스토리지(LocalStorage)에 저장되므로, 새로고침 후에도 등록 데이터가 유지됩니다.

### 기능5) 좌석 비교
* '좌석 비교 모드'를 켜고 최대 3개의 좌석을 멀티 선택할 수 있습니다.
* 하단에 나타나는 비교 바를 통해 비교하기를 누르면, 선택된 좌석들의 시야 사진과 평점, 구역 정보가 화면 분할(Split View)로 나타나 한눈에 비교 분석할 수 있습니다.

## 🚀 GitHub Pages 배포 가이드

본 프로젝트는 순수 정적 파일들로만 이루어져 있어 GitHub에서 제공하는 무료 웹 호스팅인 **GitHub Pages**로 1분 만에 웹서비스를 띄울 수 있습니다.

1. **GitHub 저장소 생성**: GitHub에 로그인 후 Public 레포지토리를 생성합니다. (예: `showview`)
2. **파일 업로드**:
   * 저장소의 `Add file` -> `Upload files`를 누릅니다.
   * 바탕화면 `sw` 폴더 내의 모든 파일(`index.html`, `styles.css`, `data.js`, `app.js`, `README.md` 및 이미지 파일들)을 업로드 창에 드래그하여 올린 뒤 **Commit changes**를 누릅니다.
3. **Pages 배포 설정**:
   * 저장소 상단 탭의 **Settings** -> 왼쪽 메뉴의 **Pages** 클릭.
   * **Branch** 설정을 `None`에서 **`main`**(또는 `master`)으로 바꾼 후 **Save** 버튼을 누릅니다.
   * 30초 정도 뒤에 상단에 생성되는 링크(`https://아이디.github.io/showview/`)로 바로 접속이 가능합니다.
